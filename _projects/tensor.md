title: Tensor Decomposition

description: This project studies community detection in multilayer networks using a probabilistic framework based on the Mixture Multilayer Stochastic Block Model (MMSBM). The approach represents multilayer graphs as adjacency tensors and applies regularized tensor decomposition to identify global and local community structures. The TWIST algorithm is used to recover node memberships and classify network layers. The methodology is illustrated through an application to a worldwide food trading network dataset.
