description: >
  This project was developed for the Big Data Technologies 2023-2024 course as part
  of the master's program in Data Science at the University of Trento. The focus is
  on implementing a Predictive Maintenance System for industrial machinery using
  real-time data collection, machine learning, and big data processing to predict
  maintenance needs, preventing unexpected downtimes, reducing costs, and optimizing efficiency.

  The system leverages IoT sensors to monitor critical machine parameters
  (e.g., temperature, pressure, vibration) and uses these data streams to predict
  when maintenance is necessary, thereby increasing uptime and reducing unplanned maintenance.
