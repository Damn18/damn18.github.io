Hackathon – Data4Econ 2026

This project was developed during the Data4Econ Hackathon 2026, hosted by Unibz, sponsored by Key to Energy under the patronage of y-SIS and the CLAssification and Data Analysis Group. The full-day event focused on forecasting hourly wind energy production by combining 1.5 years of data with weather predictions and evaluating the reliability of the input information. The challenge required the development of predictive models and the delivering of both a numerical forecast and a report explaining the methodology applied.

Big Data Technologies

This project was developed for the Big Data Technologies 2023-2024 course as part of the master’s program in Data Science at the University of Trento. The focus is on implementing a Predictive Maintenance System for industrial machinery using real-time data collection, machine learning, and big data processing to predict maintenance needs, preventing unexpected downtimes, reducing costs, and optimizing efficiency. The system leverages IoT sensors to monitor critical machine parameters (e.g., temperature, pressure, vibration) and uses these data streams to predict when maintenance is necessary, thereby increasing uptime and reducing unplanned maintenance.

Deep Learning Lab

This project explores few-shot adaptation techniques for a pre-trained CLIP model, developed for the Deep Learning course in the Data Science Master’s program at the University of Trento. The goal is to improve performance on target classes with limited labeled examples while preserving generalization to unseen classes. Three approaches are implemented and evaluated: confidence-based thresholding, anomaly detection, and a method based on Topological Data Analysis. The repository includes the experimental implementation and a comparison of the strengths and limitations of the proposed methods.

The goal was to predict hourly energy production for January 2025 using historical production data and weather forecasts. Tensor Decomposition

This project studies community detection in multilayer networks using a probabilistic framework based on the Mixture Multilayer Stochastic Block Model (MMSBM). The approach represents multilayer graphs as adjacency tensors and applies regularized tensor decomposition to identify global and local community structures. The TWIST algorithm is used to recover node memberships and classify network layers. The methodology is illustrated through an application to a worldwide food trading network dataset.

Geospatial Analysis

This project investigates the determinants of Airbnb prices in Bologna using spatial econometric models. A dataset was constructed from InsideAirbnb sources and enriched with geographic and host-related variables. After testing for spatial autocorrelation with Moran’s I, a Spatial Error Model (SEM) was estimated to account for spatial dependence. The results show that proximity to the city center and host professionalization significantly increase prices, while private/shared rooms and higher review counts are associated with lower prices.

Computational Social Science

This project analyzes the structural and content dynamics of the Bluesky social network using a dataset of one million posts. A Social Network Analysis (SNA) examines interaction patterns, centrality measures, and community structures within the reply network. In parallel, topic modeling with BERTopic identifies the main themes discussed on the platform. The results highlight a sparse network with highly influential users and fragmented topic communities.
